# WebPersonal - Portfolio de Notsxbas

Mi portafolio personal donde muestro mis proyectos, habilidades y información de contacto.

## 📁 Estructura del Proyecto

```
WebPersonal/
├── index.html           # Página principal
├── projects.html        # Página de proyectos
├── contact.html         # Página de contacto
├── styles.css           # Estilos principales
├── script.js            # JavaScript
├── CV/
│   └── SebastianCampoCV.pdf   # Mi CV
└── README.md            # Este archivo
```

## 🚀 Tecnologías Utilizadas

- **HTML5** - Estructura
- **CSS3** - Estilos y animaciones
- **JavaScript** - Interactividad
- **Google Fonts** - Typography

## ✨ Características

- Diseño moderno y minimalista
- Animaciones suaves al hacer scroll
- Botones interactivos con efectos
- Página responsive (adaptada a móviles)
- Secciones de proyectos, habilidades y contacto
